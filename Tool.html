Tool

We are conducting a report on making VR games and discussing and sharing opinions. And make the detailed information into a website. We set up a group Git repository to rely on GitHub for this project, and then invite each member's personal GitHub and give permission to edit and upload. During the production process, each member contributed their own ideas, shared their personal opinions, and finally determined the name of the game is "End", and completed the assigned task and uploaded it to the git repository, which contains these sections : Team Profile, Tools, Project Description, Skills and Jobs, Feedback, Group Reflection. But since we didn't know that we had to upload at the beginning to show the progress of our work, the Git repository reflects that our work is all for a period of time and is close to the submission time. At first, we always let one person perform the whole file was uploaded, but we changed it at a later time.
